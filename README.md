# 🚗 Colorado Motor Vehicle Sales Data Analysis

## 📌 Project Overview
This project analyzes motor vehicle sales data from 2008–2015.
The objective is to identify trends, seasonal patterns, and build a machine learning model.

## 📊 Dataset
- 501 records
- Columns: year, quarter, county, sales
- No missing values

## 🤖 Machine Learning Model
- Model: Random Forest Regressor
- RMSE ≈ 20 Million
